> 常用参数
```
-D  删除ssh-agent中的所有密钥
-d  从ssh-agent中的删除密钥
-l  显示ssh-agent中的密钥
-L  显示ssh-agent中的公钥
-X  对ssh-agent进行解锁
-x  对ssh-agent进行加锁
```

* 把专用密钥添加到`ssh-agent`的高速缓存中
```
ssh-add /key_path
```

* 查看`ssh-agent`中的密钥
```
ssh-add -l
```

* 删除`ssh-agent`中的密钥
```
ssh-add -d /key_path
```

* 连接服务器
```
ssh -v -i /Users/yb007/Desktop/CRMEB/22-xiaoshuo/key/testlive.pem root@81.70.181.86 -p 51932
```
* 私钥权限设定
```
chmod 600 /Users/yb007/Desktop/CRMEB/22-xiaoshuo/key/testlive.pem
```
