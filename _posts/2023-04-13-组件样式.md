**父类**

```
// template
<rk-xxx :suffixStyle="suffixStyle"></rk-xxx>

// js-data
suffixStyle:{
    'width':'auto',
    'color':'#969696',
    'font-size':'13px',
},

```

**子组件**

>注意子组件接收`style`要用`[]`否则微信小程序内不会生效  

```
// template
<view class="right-box" :style="[suffixStyle]"></view>
```
